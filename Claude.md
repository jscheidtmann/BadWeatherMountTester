# Bad Weather Mount Tester - Development Rules

You should be able to install this software from PyPi or by downloading an install drop from its GitHub page. 
The program should be running on Windows, Mac, or Linux. One should especially be able to run it from a Raspberry Pi. 
The project is internationalized and supports out of the box German, English and French.

Use README.md repeatedly to remind yourself of what we're going to build. 

We use uv for dependency management and hatchling for building.
